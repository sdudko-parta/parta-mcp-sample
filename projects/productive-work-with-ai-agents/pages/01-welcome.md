# Welcome

This course is about getting real work done with AI coding agents — not the demo kind, the kind you ship behind. By the end you'll have a feel for when an agent helps, when it gets in the way, and how to set it up so the first answer is usually the right one.

![Course cover](../assets/01-welcome.png)

## What you'll come away with

- A working mental model of what an agent is (and isn't).
- A repeatable way to brief an agent so it makes your trade-offs.
- Habits for catching the failure modes that cost real time.
- A short checklist you can paste at the start of any new project.

## Who this is for

Engineers who already write code by hand and have used a coding assistant a few times. We won't relitigate basics; we'll focus on the workflow.

## How the course works

Ten short pages, each with a small exercise. Don't binge it — try the exercises against your real codebase between pages.
